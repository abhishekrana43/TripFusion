# ✈️ TripFusion - Multi-Agent AI Travel Planning System

TripFusion is an AI-powered travel planning platform built using LangGraph and multiple specialized AI agents that collaborate to generate complete travel plans. Instead of relying on a single LLM call, TripFusion orchestrates multiple intelligent agents that independently research flights, hotels, and local attractions before synthesizing everything into a personalized travel itinerary.

The system demonstrates Agentic AI workflows, state management, tool calling, web search integration, and graph-based orchestration using LangGraph.

# 🧠 Agent Architecture

TripFusion consists of four specialized AI agents:

### ✈️ Flight Agent

Responsibilities:

* Search flights for selected destination
* Find suitable airlines
* Compare travel options
* Suggest departure and return schedules
* Optimize for convenience and travel preferences

Output:

```json
{
  "flight_options": [
    {
      "airline": "Example Airline",
      "departure": "09:00 AM",
      "arrival": "01:00 PM"
    }
  ]
}
```

---

### 🏨 Hotel Agent

Responsibilities:

* Research hotels near destination
* Suggest accommodations based on budget
* Evaluate ratings and amenities
* Recommend best stay options

Output:

```json
{
  "hotels": [
    {
      "name": "Grand Hotel",
      "rating": 4.8,
      "price": "$120/night"
    }
  ]
}
```

---

### 🗓️ Itinerary Agent

Responsibilities:

* Create day-by-day plans
* Suggest attractions
* Recommend restaurants
* Schedule activities
* Optimize sightseeing routes

Output:

```json
{
  "itinerary": [
    {
      "day": 1,
      "activities": [
        "City Tour",
        "Museum Visit"
      ]
    }
  ]
}
```

---

### 🤖 Final Response Agent

Responsibilities:

* Collect outputs from all agents
* Merge travel information
* Create final travel report
* Generate user-friendly recommendations

Output:

```text
Flight Details
Hotel Recommendations
Day-wise Itinerary
Travel Tips
Estimated Budget
```

# 🔄 Workflow

```text
User Request
      │
      ▼
LangGraph Router
      │
 ┌────┼────┐
 ▼    ▼    ▼
Flight Hotel Itinerary
Agent  Agent  Agent
 └────┼────┘
      ▼
Final Response Agent
      ▼
Travel Plan Delivered
```

---

# 🏗️ System Flow

Step 1:
User enters travel requirements.

Example:

```text
Plan a 5-day trip to Bali under $2000.
```

Step 2:
LangGraph initializes the workflow.

Step 3:
Flight Agent researches transportation options.

Step 4:
Hotel Agent researches accommodations.

Step 5:
Itinerary Agent generates activities.

Step 6:
All outputs are stored in shared graph state.

Step 7:
Final Response Agent consolidates results.

Step 8:
Complete travel plan is displayed.

# 🛠️ Installation

Clone Repository

```bash
git clone https://github.com/yourusername/TripFusion.git

cd TripFusion
```

Create Virtual Environment

```bash
python -m venv langgraph_env
```

Activate Environment

Windows

```bash
langgraph_env\Scripts\activate
```

Mac/Linux

```bash
source langgraph_env/bin/activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Run Application

```bash
streamlit run app.py
```
